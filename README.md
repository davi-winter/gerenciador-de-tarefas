<strong>Pré-requisitos e instalação:</strong>
1. Instalar o Visual Studio 2026
2. Instalar o BD MySQL
3. No Visual Studio, acessar Package Manager Console e inserir a instrução de construção do BD (comando "Update-Database")

<strong>Tech Stack:</strong>
- Linguagem: C#
- Back-end: ASP.NET Razor Pages com Entity Framework
- Banco de dados: MySQL
- Front-end: HTML, CSS e Javascript
- Arquitetura: MVC
